# rethinkdb-health

Tiny cli program to check the health of rethinkdb.

## Installation and Usage

```bash
$ git clone && cd ./rethinkdb-health
$ shards build
$ ./bin/rethinkdb-health && echo "your rethinkdb deployment is A-OK"
```
